CRUISE Backend API

Available endpoints:

  • /api/v1/status GET
    No parameters
    Returns a JSON object with the status of ElasticSearch and the API.
  • /api/v1/indices GET
    No parameters
    Returns a JSON object with the list of indices in ElasticSearch.
  • /api/v1/search POST
    • query: string
      The query string to search for in ElasticSearch.
    • es_index: string
      The name of the ElasticSearch index.
    • es_top_k: integer
      Top k first results to return.
    Returns a JSON object with the search results from ElasticSearch and their status.
  • /api/v1/add_docs POST
    • docs: JSON
      List with documents that you want to add to ElasticSearch.
    • index_name: string
      The name of the ElasticSearch index.
    Returns a JSON object with the status of the operation: 'OK' if added successfully.
  • /api/v1/concept_search/create_lexical_index POST
    • index_name: string
      The name of the ElasticSearch index.
    Returns a JSON object with the status of the operation: 'OK' if created successfully.
  • /api/v1/concept_search/add_concepts POST
    • concepts: JSON
      List with cocnepts that you want to add to ElasticSearch.
    • index_name: string
      The name of the ElasticSearch index.
    Returns a JSON object with the status of the operation: 'OK' if added successfully.
  • /api/v1/concept_search/lexical_search POST
    • query: string
      The query string to search for in ElasticSearch.
    • index_name: string
      The name of the ElasticSearch index.
    Returns a JSON object with the search results from ElasticSearch and their status.
